The Claude Spaces Model

The Claude Spaces Model repository documents the overall pattern — how to think about designing these workspaces, what components they should include, and best practices for structuring CLAUDE.md files. If you're interested in building your own Claude Space, start there.

The Claude Space Self-Ideator is a meta-space that helps you brainstorm and design new Claude Spaces, using Claude itself to identify workflows that would benefit from this pattern.
