Non-Code Uses of Claude Code

Claude Code was built as a coding assistant. But the moment you realise it's an AI agent with filesystem access, shell execution, and MCP integrations, the use cases expand far beyond writing software. This index documents over a hundred repositories that explore what happens when you point Claude Code at problems that have nothing to do with programming.

The Non-Code Claude Code repository captures the broader philosophy: Claude Code as a general-purpose AI workspace that happens to run in a terminal.

Legal Research and Case Management

The terminal turns out to be a surprisingly effective interface for legal work. The Legal category includes workspaces for tracking cases, analysing evidence, and building forensic timelines.

The Claude Case File workspace provides structured case management with Claude handling document organisation, timeline construction, and cross-referencing. Claude Evidence Assistant focuses specifically on evidence handling and chain-of-custody documentation. And Claude Code Lawyer provides a more general legal research interface.

The Claude Legal Aid Clinic explores an interesting angle: using Claude as a first-pass legal aid tool that can help people understand their rights and options before engaging professional counsel.

Health and Wellbeing

Several workspaces in the Health & Wellbeing category use Claude to manage personal health data. The Claude Health Helper provides a general health documentation workspace, while Claude Therapy Tracker focuses specifically on therapy session notes and progress tracking.

These workspaces highlight an advantage of the terminal-based approach: your health data stays local on your machine. There's no cloud service, no account, no third-party data processing. Claude works with files on your filesystem, and nothing leaves your computer unless you explicitly push it somewhere.

Financial Planning

The Financial Planning category includes the Claude Budget Workspace Template for personal budgeting and the Claude Purchasing Assistant for evaluating purchase decisions. Claude can parse bank statements, categorise expenses, and help with tax preparation — all from structured files on your local disk.

Communications and Writing

The Communications & Writing category covers content creation, blog management, and media strategy. The Claude Blog Manager turns Claude into a content management system, while Claude Media Monitor and the Claude News Fetcher handle media monitoring workflows.

The Claude Code Writing Squad demonstrates multi-agent collaboration for writing projects — different Claude agents handling research, drafting, editing, and fact-checking as a coordinated team.

Research and Analysis

The Research category is one of the largest in the index. The Claude Deep Research Template provides a structured approach to multi-source research, while the Claude Stack Research Workspace focuses on technology evaluation.

Claude Dork generates search queries (dorks) across multiple platforms for OSINT-style research. Claude OSINT Investigator takes this further with a full open-source intelligence workspace.

Productivity and Decision-Making

The Productivity category includes tools for personal planning and decision frameworks. Claude Decision Evaluation Framework provides structured decision analysis. Claude Task Manager is a full task management system running in the terminal. Claude Diary Planner Template combines journaling with daily planning.

The Pattern

What connects all of these is a simple pattern: define a workspace with structured context (CLAUDE.md), give Claude access to the right tools (filesystem, MCP servers, slash commands), and let it manage workflows that would traditionally require dedicated GUI applications. The terminal becomes a universal interface for AI-assisted work.

If you're interested in this approach, the Claude Spaces Model repository documents the methodology, and the Claude Spaces page explains the workspace pattern in detail.
